*************
Configuration
*************

The *EDocument Peppol Module* uses values from settings in the
``[edocument_peppol]`` section of the :ref:`trytond:topics-configuration`.

.. _config-edocument_peppol.filestore:

``filestore``
=============

The ``filestore`` defines if the Peppol documents are stored in the
:ref:`FileStore <trytond:ref-filestore>`.

The default value is: ``True``

``store_prefix``
================

The ``store_prefix`` contains the prefix to use with the :ref:`FileStore
<trytond:ref-filestore>`.

The default value is: ``None``
