.. _Configure Peppyrus credentials:

Configuration Peppyrus credentials
==================================

First you must `register with Peppyrus
<https://customer.peppyrus.be/register>`_, `create your participant ID
<https://customer.peppyrus.be/customer/organization/peppol/identifier>`_.
You can choose to receive the document types :guilabel:`Peppol BIS Billing UBL
Invoice V3` and :guilabel:`Peppol BIS Billing UBL Credit Note V3`.

Then create an API key and copy it as :guilabel:`API Key` on the `Peppyrus
Service <edocument_peppol:Setting Peppol services>`.
