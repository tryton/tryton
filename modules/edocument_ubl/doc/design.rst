The *EDocument UBL Module* introduces the following concepts:

.. _model-edocument.ubl.invoice:

UBL Invoice
===========

The *UBL Invoice* renders an `Invoice <account_invoice:model-account.invoice>`
into an :abbr:`XML (eXtensible Markup Language)` file that conforms to the
`UBL-Invoice-2.4
<https://docs.oasis-open.org/ubl/os-UBL-2.4/mod/summary/reports/UBL-Invoice-2.4.html>`_
standard.
